## CheckFileExtension

1) 파일유형 제한 설정

   허용확장자, 차단확장자를 설정하여 파일유형별로 필터링이 가능하다.
   
2) 압축파일 내에 파일유형 제한 기능

   압축파일 내부에 있는 파일 리스트에서 파일유형별로 필터링이 가능하다.

참고사항:
1) 파일 확장자 외에 파일 시그니처를 통해 파일을 인식해야 한다.
 
2) 압축 파일 내에 허용되지 않은 파일 유형이 포함된 경우 허용하지 않아야 한다
